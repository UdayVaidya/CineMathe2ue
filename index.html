API Routes
All backend REST endpoints — base: /api
Auth — /api/auth
Method Path Description Access
POST /api/auth/register Register new user public
POST /api/auth/login Login (role required in body) public
POST /api/auth/logout Clear JWT cookie private
GET /api/auth/me Get current user private
User — /api/users
Method Path Description Access
GET /api/users/profile Get user profile private
PUT /api/users/profile Update username, avatar private
GET /api/users/favorites Get all favorites private
POST /api/users/favorites Add to favorites private
DEL /api/users/favorites/:tmdbId Remove from favorites private
GET /api/users/history Get watch history private
POST /api/users/history Add to history (max 50) private
DEL /api/users/history Clear all watch history private
Admin — /api/admin
Method Path Description Access
GET /api/admin/movies Get all custom movies admin
POST /api/admin/movies Create a movie admin
PUT /api/admin/movies/:id Update movie details admin
DEL /api/admin/movies/:id Delete a movie admin
GET /api/admin/users Get all users admin
PUT /api/admin/users/:id/ban Toggle ban status admin
DEL /api/admin/users/:id Delete a user admin
